# Keyboard reference

Shortcuts use `Cmd` on macOS and `Ctrl` on Windows and Linux.

## Global

| Action | Shortcut |
|---|---|
| Open global search | `Cmd+K` / `Ctrl+K` |
| Quick open file | `Cmd+P` / `Ctrl+P` |
| Toggle sidebar | `Cmd+B` / `Ctrl+B` |
| Toggle chat panel | `Cmd+J` / `Ctrl+J` |
| Navigate to chat | `Cmd+Shift+L` / `Ctrl+Shift+L` |
| Toggle terminal panel | `Ctrl+\`` |
| Open keyboard shortcuts modal | `?` |

## Editor (files view)

| Action | Shortcut |
|---|---|
| Save file | `Cmd+S` / `Ctrl+S` |

## Chat composer

| Action | Shortcut |
|---|---|
| Send message | `Enter` |
| Insert a newline (multi-line input) | `Shift+Enter` |
| Open model selector | `Cmd+Shift+M` / `Ctrl+Shift+M` |
| Navigate slash-command menu down | `Arrow Down` (while menu is open) |
| Navigate slash-command menu up | `Arrow Up` (while menu is open) |
| Dismiss slash-command menu | `Escape` (while menu is open) |

## Search modal

| Action | Shortcut |
|---|---|
| Open / close search | `Cmd+K` / `Ctrl+K` |
| Close search | `Escape` |
