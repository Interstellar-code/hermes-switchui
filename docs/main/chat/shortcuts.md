# Keyboard shortcuts

Switch UI supports several keyboard shortcuts in the chat view. Most are available whenever the composer is focused.

> [SCREENSHOT: Chat view with composer focused, illustrating the keyboard shortcut overlay or annotated screenshot]

## Composer shortcuts

| Shortcut | Action |
|----------|--------|
| **Enter** | Send the current message |
| **Shift+Enter** | Insert a newline without sending |
| **Cmd+Enter** / **Ctrl+Enter** | Send the current message (alternative to Enter) |
| **Cmd+Shift+M** / **Ctrl+Shift+M** | Open the model selector |
| **/** (at start of input) | Open the slash command menu |
| **Arrow Up / Down** | Navigate slash command menu when open |
| **Escape** | Dismiss the slash command menu |

## Navigation shortcuts

These work anywhere in the chat view, not just inside the composer:

| Shortcut | Action |
|----------|--------|
| **Escape** | Close modals, drawers, and menus |

## Notes

- On macOS, **Cmd** is the Command key. On Windows and Linux, use **Ctrl** in its place.
- **Enter** alone sends the message. If you want a multi-line prompt, use **Shift+Enter** for each line break before the final **Enter** to send.
- The model selector shortcut (**Cmd+Shift+M** / **Ctrl+Shift+M**) works even when the composer is not focused, as long as the chat screen is active.
