Hermes Switch UI — Docs Authoring Pipeline

How a markdown file becomes a rendered page in /docs/$

docs-manifest.yaml

  • · Central slug registry — one entry per doc
  • · Controls sidebar display order
  • · Title override: manifest title beats frontmatter
  • · Files not in manifest are silently ignored

Render Pipeline (docs-render.ts)

  • · remark-gfm → remark-rehype → rehype-stringify
  • · rehype-pretty-code: syntax highlighting
  • · rehypeMermaid: fenced ```mermaid → <pre class="mermaid">
  • · rewriteDocImages → /api/docs-asset?path=…
  • · rewriteDocLinks → /docs/<slug> (in-app nav)

/api/docs-asset

  • · Serves images: .png .jpg .gif .webp .svg
  • · Serves HTML diagrams: .html .htm
  • · HTML: CSP script-src 'none' + X-Frame-Options: SAMEORIGIN
  • · Path traversal protection — no .. allowed
  • · Requires session authentication
