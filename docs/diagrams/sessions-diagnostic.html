Sessions Diagnostic Tree

Decision tree: identify your session symptom → follow to the fix

Session disappears

  • • Agent not running → start agent, reload page
  • • Wrong gateway URL → Settings → Agent → Reconnect
  • • Capability probe failed → check console for [gateway] log
  • • Stale cache → hard refresh (Cmd+Shift+R)

Delete fails / reappears

  • • Kanban DB ISO timestamp poisoning bug
  • • Direct DB write bypasses int coercion → 503 errors
  • • Fix: restart agent process
  • • Persists? Restore ~/.hermes/kanban.db.bak.*

All sessions gone

  • • sessions.db corrupted after agent crash
  • • Check ~/.hermes/ for .bak backup files
  • • Copy most recent backup → sessions.db → restart
  • • No backup: accept fresh start, data is lost
