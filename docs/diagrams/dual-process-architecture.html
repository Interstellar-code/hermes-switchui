Hermes Switch UI — Dual-Process Architecture

Browser-based shell (port 3000) paired with the Hermes Agent gateway (port 8642)

Switch UI (port 3000)

  • • Browser-based TanStack Start app
  • • Auth-gated workspace UI
  • • Streams responses via SSE
  • • Persists nothing locally (state in agent)

Hermes Agent (port 8642)

  • • Python gateway runtime
  • • Owns sessions, memory, skills, MCP
  • • Routes prompts to chosen provider
  • • Runs workflow DAGs and tools

~/.hermes/ storage

  • • Sessions DB + memory files
  • • Installed skills + MCP servers
  • • Provider config + API keys
  • • Survives restarts and crashes
